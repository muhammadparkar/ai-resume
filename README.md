## Introduction

AI Resume Analyzer is a modern web application that evaluates resumes against job descriptions using AI-powered insights. Built with React, React Router, and Puter.js, the platform offers seamless authentication, secure resume storage, and intelligent resume-to-job matching ‚Äî all without requiring a traditional backend.

Users can upload multiple resumes, compare them with job listings, receive ATS compatibility scores, and get personalized feedback to improve their chances of selection. The application features a clean, reusable UI designed for scalability and maintainability.

Puter is an open-source, privacy-first internet operating system that acts as a personal cloud.

Puter.js is a lightweight client-side SDK that provides:
- Serverless authentication
- File storage
- Database access
